## 수입을 늘리기 전에, 새는 돈부터 막자

매달 월급이 들어오면 자동으로 빠져나가는 돈들이 있습니다. 통신비, 보험료, 구독 서비스, 관리비... 이런 **고정지출**은 한 번만 손보면 매달 자동으로 절약됩니다. 실제로 점검해보면 **월 30~50만 원**은 줄일 수 있습니다.

## 1. 통신비: 알뜰폰으로 월 5만원 절약

대형 통신사 요금제를 쓰고 있다면 **알뜰폰(MVNO)**으로 바꾸는 것만으로 큰 절약이 됩니다.

| 구분 | 대형 통신사 | 알뜰폰 |
|------|-----------|--------|
| 월 요금 | 6~8만 원 | 1~3만 원 |
| 데이터 | 무제한 | 11GB~무제한 |
| 통화 | 무제한 | 무제한 |

**절약액: 월 4~5만 원**

- 추천: 리브모바일, 핀다이렉트, 여유알뜰모바일
- 기존 번호 그대로 이전 가능
- 가족 결합 할인까지 알뜰폰끼리 가능한 곳도 있음

## 2. 보험료: 중복·과잉 보장 정리로 월 10~20만원 절약

한국인 평균 보험료는 월 **30만 원 이상**입니다. 하지만 대부분 중복 보장과 불필요한 특약이 포함되어 있습니다.

**점검 포인트:**
- 실비보험이 2개 이상? → 1개로 충분 (중복 청구 불가)
- 암보험 + 종합보험 암 특약 → 중복 확인
- 저축성 보험 → 수익률 확인 후 해지 검토
- 운전자보험 특약 → 자동차보험과 겹치는지 확인

**절약액: 월 10~20만 원**

> 보험다모아(e-insmarket.or.kr)에서 내 보험을 한눈에 조회할 수 있습니다.

## 3. 구독 서비스: 안 쓰는 구독 정리로 월 3~5만원 절약

넷플릭스, 유튜브 프리미엄, 멜론, 쿠팡 로켓와우, 네이버 플러스... 하나하나는 작지만 합치면 큽니다.

**점검 방법:**
1. 카드 명세서에서 정기결제 항목 전체 확인
2. 최근 1개월간 실제 사용한 서비스만 남기기
3. 가족/친구와 공유 가능한 서비스는 공유

**꿀팁:**
- 넷플릭스 + 티빙 → 번갈아 구독 (한 달씩 교대)
- 유튜브 프리미엄 → 가족 요금제로 5명 분할
- 음악 → 유튜브 뮤직(프리미엄에 포함)으로 통합

**절약액: 월 3~5만 원**

## 4. 관리비: 에너지 절약으로 월 3~5만원 절약

관리비 고지서를 자세히 보면 **전기, 가스, 수도** 각각의 사용량을 확인할 수 있습니다.

**실천법:**
- 대기전력 차단 멀티탭 사용 (연간 5만 원 절약)
- 난방 온도 1도 낮추기 (가스비 7% 절감)
- LED 조명으로 교체 (전기료 40% 절감)
- 세탁기 찬물 세탁 + 모아서 돌리기
- 빨래 건조기 대신 자연 건조

**절약액: 월 3~5만 원**

## 5. 자동차 유지비: 월 10만원 이상 절약 가능

자동차가 있다면 유지비를 점검해보세요.

- **자동차보험**: 다이렉트 보험으로 변경 (20~30% 절약)
- **주유**: 주유 할인 카드 + 알뜰주유소 앱 활용
- **세차**: 셀프세차로 전환 (월 3만 원 → 5천 원)
- **정비**: 카센터 대신 공업사 활용 (부품비 40% 절약)
- **주차**: 월주차 계약 vs 시간제 비교

**절약액: 월 10~15만 원**

## 6. 카드 연회비·수수료: 불필요한 카드 정리

카드가 3장 이상이라면 점검이 필요합니다.

- 연회비 대비 실제 받는 혜택 계산
- 전월 실적 조건 못 채우는 카드 → 해지
- 체크카드로 전환 검토 (과소비 방지 효과)
- 해외 결제 수수료 무료 카드 1장만 유지

**절약액: 월 1~3만 원**

## 7. 은행 수수료: 무료 조건 활용

계좌 이체 수수료, ATM 수수료도 모이면 큽니다.

- 급여이체 계좌 등록 → 이체 수수료 면제
- 인터넷/모바일뱅킹 이용 → 수수료 할인
- 토스, 카카오뱅크 → 이체 수수료 무료

**절약액: 월 1만 원**

## 총 절약 효과 정리

| 항목 | 월 절약액 |
|------|----------|
| 통신비 | 4~5만 원 |
| 보험료 | 10~20만 원 |
| 구독 서비스 | 3~5만 원 |
| 관리비 | 3~5만 원 |
| 자동차 | 10~15만 원 |
| 카드 | 1~3만 원 |
| 은행 수수료 | 1만 원 |
| **합계** | **32~54만 원** |

## 실행 순서 추천

1. **오늘**: 카드 명세서 확인 → 구독 서비스 정리
2. **이번 주**: 알뜰폰 비교 → 통신사 변경 신청
3. **이번 달**: 보험다모아에서 보험 조회 → 중복 정리
4. **다음 달**: 자동차보험 만기 시 다이렉트 전환

## 마무리

고정지출 절약의 핵심은 **한 번만 손보면 매달 자동으로 절약된다**는 점입니다. 위 7가지를 모두 실행하면 연간 **400~600만 원** 이상을 추가로 저축할 수 있습니다. 오늘 당장 카드 명세서부터 확인해보세요!
